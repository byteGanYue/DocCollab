## 使用 CDN

你还可以通过CDN获取构建好的文件。将以下代码添加到 HTML 文件的 `<script>` 标签中：

```
<script src="https://cdn.jsdelivr.net/npm/@ni0duann/monitor-sdk@1.0.1/dist/index.min.js"></script>
```

## 使用 script 标签引入

通过直接在 HTML 文件中添加 `<script>` 标签，引入构建好的文件：

```
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <!-- 引入文件 -->
    <script src="https://cdn.jsdelivr.net/npm/@ni0duann/monitor-sdk@1.0.1/dist/index.min.js"></script>
  </head>
</html>
```
